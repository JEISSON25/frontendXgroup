### Installation

``` bash
$ npm install
```

or

``` bash
$ yarn install
```

### Basic usage

``` bash
# dev server with hot reload at http://localhost:3000
$ npm run dev  
```

or 

``` bash
# dev server with hot reload at http://localhost:3000
$ yarn run dev
```

or
### Docker and DockerCompose
run command: docker-compose up --build
